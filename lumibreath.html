Knowable – Trustpilot Review Analysis Platform

Bachelor Project · NLP · R · Data Analysis

Overview

Knowable challenges the idea that star ratings alone reflect customer experience. I built a data pipeline that scrapes Trustpilot reviews, applies NLP sentiment and topic modelling, and presents deeper insights into what customers actually say.

My Role

  • Concept development
  • Data pipeline development in R
  • Web scraping & data cleaning
  • Sentiment and topic modelling
  • Prototype of insight presentation

Tools & Methods

R NLP Sentiment Analysis Topic Modelling Web Scraping

Outcome

The project demonstrated how text-based review analysis reveals deeper insights than ratings, supporting smarter decisions for both companies and consumers.
